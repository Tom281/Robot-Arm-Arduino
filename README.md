# Robot-Arm-Arduino
## Controls:
- |\<\< : Select swivel servo
- \>\>| : Select height servo
- \>|| : Select clamp servo
- EQ : Select reach servo
- \- : Decrease selected servo angle
- \+ : Increase selected servo angle
- 0 : Put arm in default position
- 1 : Extend arm
- 2 : Retract arm
- 4 : Clamp arm
- 5 : Unclamp arm
- 7 : Swivel left
- 8 : Swivel center
- 9 : Swivel right
